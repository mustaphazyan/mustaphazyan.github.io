# Melody Music Player - مشغل الموسيقى الاحترافي

تطبيق ويب عصري لتشغيل الموسيقى بتصميم مستوحى من Spotify، يدعم اللغة العربية، ويتميز بواجهة مستخدم زجاجية (Glassmorphism) وتجربة مستخدم سلسة.

## 🚀 الميزات
- **واجهة احترافية:** تصميم Dark Mode متجاوب مع جميع الشاشات.
- **نظام إضافة سهل:** أضف أغانيك بمجرد تعديل جدول HTML في ملف `index.html`.
- **تحكم كامل:** تشغيل، إيقاف، تخطي، عشوائي، وتكرار.
- **محلل صوتي بصري:** رسوم متحركة تتفاعل مع الموسيقى في وضع الشاشة الكاملة.
- **إشعارات:** تظهر عند تغيير الأغنية.
- **بحث ذكي:** ابحث عن أغانيك المفضلة بالاسم أو الفنان.
- **اختصارات لوحة المفاتيح:** مسافة للتشغيل/الإيقاف، والأسهم للتقديم والتأخير.

## 📁 هيكل المجلدات
```
/project
  ├── index.html        # الصفحة الرئيسية والمشغل
  ├── music.html        # صفحة المكتبة
  ├── style.css         # التنسيقات والجماليات
  ├── script.js        # منطق التشغيل والميزات
  ├── /music            # ضع ملفات mp3 هنا
  └── /covers           # ضع صور الأغلفة هنا
```

## 🛠️ كيف تضيف أغانيك الخاصة؟
1. ضع ملف الأغنية (مثلاً `song.mp3`) في مجلد `music/`.
2. ضع صورة الغلاف (مثلاً `image.jpg`) في مجلد `covers/`.
3. افتح ملف `index.html` وابحث عن الوسم `<tbody id="raw-song-data">`.
4. أضف سطراً جديداً بهذا الشكل:
```html
<tr>
    <td>اسم الأغنية</td>
    <td>song.mp3</td>
    <td>اسم الفنان</td>
    <td>image.jpg</td>
</tr>
```

## 🌐 النشر على GitHub Pages
1. ارفع جميع الملفات إلى مستودع (Repository) جديد على GitHub.
2. اذهب إلى **Settings** في المستودع.
3. اختر **Pages** من القائمة الجانبية.
4. تحت قسم **Build and deployment**، تأكد من اختيار فرع `main` ثم اضغط **Save**.
5. سيظهر لك رابط الموقع بعد دقائق قليلة.
